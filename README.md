# MODE AR

تطبيق تحويل فوري بين العربية والإنجليزية

## المميزات
- تحويل فوري بضغطة واحدة (Ctrl + Q)
- يعمل في الخلفية
- آمن ومجاني 100%
- محسن للجوال والكمبيوتر

## التحديث الأخير
- تم تحسين الموقع بالكامل للجوال
- إضافة responsive design
- قائمة جوال محسنة
- زر تحميل ثابت
